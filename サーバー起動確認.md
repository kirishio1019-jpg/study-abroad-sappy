# 開発サーバー起動確認

## 起動状況の確認方法

### 方法1: ブラウザで確認
以下のURLをブラウザで開いてください：
```
http://localhost:3000
```

### 方法2: テストページで確認
以下のURLを開いて、テストページが表示されるか確認してください：
```
http://localhost:3000/test.html
```

---

## サーバーが起動している場合

✅ **正常に起動している場合:**
- `http://localhost:3000` でページが表示されます
- または、`http://localhost:3000/test.html` でテストページが表示されます

---

## サーバーが起動していない場合

❌ **まだ起動していない場合:**
以下の手順で手動で起動してください：

### ステップ1: Cursorのターミナルを開く
- `Ctrl + Shift + ` (バッククォート) を押す
- または、上部メニューから「ターミナル」→「新しいターミナル」

### ステップ2: コマンドを実行
ターミナルに以下を入力してEnterキーを押してください：
```powershell
cd C:\Users\kiris\.cursor\study-abroad-comparison
npm run dev
```

### ステップ3: 起動メッセージを確認
以下のようなメッセージが表示されるまで待ってください：
```
✓ Ready in X seconds
○ Local:        http://localhost:3000
```

### ステップ4: ブラウザで開く
表示されたURL（`http://localhost:3000`）をブラウザで開いてください。

---

## エラーが出る場合

### エラー1: "Port 3000 is already in use"
→ ポート3000が使用中です。別のポートを使用：
```powershell
npm run dev -- -p 3001
```
その後、`http://localhost:3001` でアクセス

### エラー2: "Cannot find module"
→ 依存関係をインストール：
```powershell
npm install
```
その後、再度 `npm run dev` を実行

### エラー3: "npm が見つかりません"
→ Node.jsがインストールされていません
- Node.jsをインストールしてください: https://nodejs.org/

---

## サーバーを停止する方法

- ターミナルで `Ctrl + C` を押す
- または、ターミナルウィンドウを閉じる
