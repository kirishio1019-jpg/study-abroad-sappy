# 🚀 今すぐサーバーを起動する方法

## 問題：開発サーバーが起動していません

「このサイトにアクセスできません」と表示されるのは、開発サーバーが起動していないためです。

---

## ✅ 解決方法：開発サーバーを起動する

### 方法1: バッチファイルをダブルクリック（最も簡単！）

1. **ファイルエクスプローラーを開く**
   - Windowsキー + E を押す

2. **以下のフォルダに移動：**
   ```
   C:\Users\kiris\.cursor\study-abroad-comparison
   ```

3. **`開発サーバー起動.bat` というファイルを探す**

4. **そのファイルをダブルクリック**

5. **黒い画面（コマンドプロンプト）が開きます**

6. **30秒〜1分待つ**

7. **以下のようなメッセージが表示されます：**
   ```
   ✓ Ready in X seconds
   ○ Local:        http://localhost:3000
   ```

8. **このメッセージが表示されたら、サーバーは起動しています！**

9. **ブラウザで `http://localhost:3000` を開く**

10. **ページが表示されれば成功！**

---

### 方法2: コマンドプロンプトを使う

1. **Windowsキーを押す**

2. **「cmd」と入力してEnterキーを押す**

3. **コマンドプロンプトが開いたら、以下を実行：**

   ```
   cd C:\Users\kiris\.cursor\study-abroad-comparison
   ```
   Enterキーを押す

4. **次に：**
   ```
   npm run dev
   ```
   Enterキーを押す

5. **30秒〜1分待つ**

6. **以下のようなメッセージが表示されます：**
   ```
   ✓ Ready in X seconds
   ○ Local:        http://localhost:3000
   ```

7. **このメッセージが表示されたら、サーバーは起動しています！**

8. **ブラウザで `http://localhost:3000` を開く**

9. **ページが表示されれば成功！**

---

## ❗ 重要なポイント

### 1. コマンドプロンプトのウィンドウは閉じない

開発サーバーを起動した後は、**コマンドプロンプトのウィンドウを閉じないでください**。

ウィンドウを閉じると、開発サーバーも停止してしまいます。

### 2. サーバーを停止したい場合

コマンドプロンプトのウィンドウで **`Ctrl + C`** を押してください。

---

## 🔍 確認方法

開発サーバーが起動しているかどうかを確認する方法：

1. **コマンドプロンプトのウィンドウを見る**
   - 「✓ Ready」というメッセージが表示されているか確認

2. **ブラウザで `http://localhost:3000` を開く**
   - ページが表示されれば、サーバーは起動しています
   - 「このサイトにアクセスできません」と表示される場合は、サーバーが起動していません

---

## 📋 手順のまとめ

1. ✅ バッチファイルをダブルクリック（またはコマンドプロンプトで `npm run dev`）
2. ✅ 「✓ Ready」というメッセージが表示されるまで待つ（30秒〜1分）
3. ✅ ブラウザで `http://localhost:3000` を開く
4. ✅ ページが表示されれば成功！

---

## ❓ うまくいかない場合

### 「npm が見つかりません」というエラー

→ Node.jsがインストールされていない可能性があります。
   まず、Node.jsをインストールする必要があります。

### 「Port 3000 is already in use」というエラー

→ ポート3000が既に使用されています。
   
   解決方法：
   ```
   npm run dev -- -p 3001
   ```
   その後、`http://localhost:3001` でアクセス

### その他のエラー

→ コマンドプロンプトに表示されたエラーメッセージを教えてください。

---

**まずは、方法1（バッチファイルをダブルクリック）を試してみてください！**
