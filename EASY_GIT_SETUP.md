# 超簡単！Git設定方法

## 🎯 Cursorのファイルエクスプローラーから直接編集

プロジェクト内に `.git/config` ファイルを作成しました！

### ステップ1: ファイルを開く

1. **Cursorの左側のファイルエクスプローラーを見る**
2. **`.git` フォルダを探す**（非表示の場合は表示する）
3. **`config` ファイルをクリック**

**もし `.git` フォルダが見えない場合：**
- `.git` フォルダは通常非表示ですが、Cursorで開けます
- ファイルエクスプローラーの検索バーで「config」と検索

---

### ステップ2: あなたの情報に置き換える

`config` ファイルを開いて、以下の部分を編集：

```ini
[user]
	name = あなたのGitHubのユーザー名
	email = あなたのメールアドレス
```

**例：**
```ini
[user]
	name = kiris
	email = b2400560@gl.aiu.ac.jp
```

### ステップ3: 保存

- `Ctrl + S` で保存

---

## ✅ これで完了！

保存したら、もう一度コミットを試してください：

1. ソース管理パネル（📊）を開く
2. コミットメッセージを入力
3. `Ctrl + Enter` を押す

---

## 🔍 ファイルが見つからない場合

もし `.git/config` ファイルが見つからない場合は、以下の方法を試してください：

### 方法1: 手動で作成

1. Cursorで **`Ctrl + N`** を押して新しいファイルを開く
2. 以下の内容をコピー＆ペースト：

```ini
[user]
	name = あなたのGitHubのユーザー名
	email = あなたのメールアドレス
```

3. あなたの情報に置き換える
4. **名前を付けて保存**（`Ctrl + Shift + S`）
5. ファイル名：`.gitconfig`
6. 保存場所：`study-abroad-comparison` フォルダ
7. 保存

### 方法2: Cursorの設定から

1. `Ctrl + ,` を押して設定を開く
2. 検索バーに「git」と入力
3. 「Git: Path」などの設定が表示されます
4. 直接入力できる設定があれば、そこから入力

---

## 📝 設定する情報

- **name**: GitHubのユーザー名、または任意の名前
  - 例：`kiris`

- **email**: GitHubに登録しているメールアドレス
  - 例：`b2400560@gl.aiu.ac.jp`

このメールアドレスはコミット履歴に表示されます。
