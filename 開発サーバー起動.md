# 開発サーバーの起動方法

## 方法1: バッチファイルを使用（最も簡単）

1. **ファイルエクスプローラーを開く**
   - `C:\Users\kiris\.cursor\study-abroad-comparison` フォルダを開く

2. **`開発サーバー起動.bat` をダブルクリック**
   - コマンドプロンプトウィンドウが開きます
   - 自動的に開発サーバーが起動します

3. **ブラウザで開く**
   - `http://localhost:3000` を開く

4. **停止する場合**
   - コマンドプロンプトウィンドウで `Ctrl + C` を押す

---

## 方法2: Cursorのコマンドパレットを使用

1. **コマンドパレットを開く**
   - `Ctrl + Shift + P` を押す

2. **「Terminal: Create New Terminal」を検索して実行**
   - または「ターミナル: 新しいターミナルを作成」と入力

3. **ターミナルが開いたら、以下を入力:**
   ```powershell
   cd C:\Users\kiris\.cursor\study-abroad-comparison
   npm run dev
   ```

---

## 方法3: Cursorのターミナルパネルを使用

1. **ターミナルパネルを開く**
   - 下部の「ターミナル」タブをクリック
   - または `Ctrl + Shift + ` (バッククォート)

2. **コマンドを入力:**
   ```powershell
   cd C:\Users\kiris\.cursor\study-abroad-comparison
   npm run dev
   ```

---

## トラブルシューティング

### バッチファイルが実行できない場合
- 右クリック → 「管理者として実行」を試してください

### 「npm が見つかりません」というエラー
- Node.jsがインストールされていない可能性があります
- または、環境変数が設定されていない可能性があります

### ポート3000が使用中
- 別のアプリケーションがポート3000を使用しています
- そのアプリを終了するか、別のポートを使用:
  ```powershell
  npm run dev -- -p 3001
  ```
