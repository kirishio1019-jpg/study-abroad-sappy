# 🚀 開発サーバーを起動する簡単な方法

## ⚡ 最も簡単な方法：バッチファイルを使う

### 方法1: バッチファイルをダブルクリック（超簡単！）

1. **ファイルエクスプローラーを開く**

2. **以下のフォルダに移動：**
   ```
   C:\Users\kiris\.cursor\study-abroad-comparison
   ```

3. **`開発サーバー起動.bat` というファイルを見つける**

4. **そのファイルをダブルクリック**

5. **黒い画面（コマンドプロンプト）が開く**

6. **「✓ Ready」というメッセージが表示されるまで待つ（30秒〜1分）**

7. **ブラウザで `http://localhost:3000` を開く**

8. **ページが表示されれば成功！**

---

## 方法2: コマンドプロンプトを使う

### ステップ1: コマンドプロンプトを開く

1. **Windowsキーを押す**

2. **「cmd」と入力**

3. **「コマンドプロンプト」をクリック**

### ステップ2: コマンドを実行

コマンドプロンプトが開いたら、以下を**順番に**実行してください：

#### 1つ目：以下の文字をコピーして貼り付け、Enterキーを押す

```
cd C:\Users\kiris\.cursor\study-abroad-comparison
```

#### 2つ目：以下の文字をコピーして貼り付け、Enterキーを押す

```
npm run dev
```

### ステップ3: 起動を待つ

「✓ Ready」というメッセージが表示されるまで待ってください（30秒〜1分）：

```
✓ Ready in X seconds
○ Local:        http://localhost:3000
```

### ステップ4: ブラウザで開く

1. **ブラウザを開く**
2. **アドレスバーに以下を入力：**
   ```
   http://localhost:3000
   ```
3. **Enterキーを押す**
4. **ページが表示されれば成功！**

---

## ❗ 重要

開発サーバーを起動した後は、**コマンドプロンプトのウィンドウを閉じないでください**。

ウィンドウを閉じると、開発サーバーも停止してしまいます。

**サーバーを停止したい場合**は、コマンドプロンプトで `Ctrl + C` を押してください。

---

## ❓ うまくいかない場合

### 「npm が見つかりません」というエラーが出る

→ Node.jsがインストールされていない可能性があります。
   まず、Node.jsをインストールする必要があります。

### バッチファイルが実行されない

→ ファイルを右クリックして、「管理者として実行」を試してみてください。

### その他のエラーが出る

→ コマンドプロンプトに表示されたエラーメッセージを教えてください。
