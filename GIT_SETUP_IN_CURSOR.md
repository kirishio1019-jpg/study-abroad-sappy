# Cursorで直接設定する方法

## ✅ 最も簡単な方法：設定ファイルを直接編集

2つのファイルを作成しました。どちらか一方を編集してください：

---

## 方法1: `.git/config` ファイルを編集（推奨）

1. **Cursorのファイルエクスプローラーを開く**
2. **`.git` フォルダを探す**
   - 非表示の場合は、検索バーで「.git」と検索
3. **`config` ファイルをクリックして開く**
4. **以下の部分を編集**：

```
[user]
	name = あなたのGitHubのユーザー名
	email = あなたのメールアドレス
```

**あなたの情報に置き換える例：**
```
[user]
	name = kiris
	email = b2400560@gl.aiu.ac.jp
```

5. **保存（`Ctrl + S`）**

---

## 方法2: `.vscode/settings.json` ファイルを編集

1. **Cursorのファイルエクスプローラーで `.vscode` フォルダを開く**
2. **`settings.json` ファイルを開く**
3. **以下の内容を編集**：

```json
{
  "git.config": {
    "user.name": "あなたのGitHubのユーザー名",
    "user.email": "あなたのメールアドレス"
  }
}
```

**あなたの情報に置き換える例：**
```json
{
  "git.config": {
    "user.name": "kiris",
    "user.email": "b2400560@gl.aiu.ac.jp"
  }
}
```

4. **保存（`Ctrl + S`）**

---

## 📝 入力する情報

- **name**: GitHubのユーザー名、または表示したい名前
  - 例：`kiris`

- **email**: GitHubに登録しているメールアドレス
  - 例：`b2400560@gl.aiu.ac.jp`

---

## ✅ 設定後

1. **Cursorを再起動**（完全に閉じて再度開く）
2. **ソース管理パネル（📊）を開く**
3. **コミットメッセージを入力して `Ctrl + Enter`**

これでコミットできるはずです！

---

## 🆘 ファイルが見つからない場合

### `.git/config` が見つからない場合：

1. ファイルエクスプローラーの検索バーで「config」と検索
2. または、`Ctrl + P` を押して「.git/config」と入力

### `.vscode/settings.json` が見つからない場合：

1. `.vscode` フォルダがなければ、作成します
2. `Ctrl + N` で新しいファイルを作成
3. 上記のJSONをコピー＆ペースト
4. `Ctrl + Shift + S` で保存
5. ファイル名：`settings.json`
6. 保存場所：`.vscode` フォルダ内

---

どちらの方法でもOKです！設定したら教えてください。
