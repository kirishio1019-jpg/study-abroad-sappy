# Gitの設定ファイルを手動で作成する方法

## 🔧 方法1: プロジェクトローカルに設定ファイルを作成（推奨）

このプロジェクトだけにGit設定を適用する場合：

### 手順

1. **プロジェクトフォルダ内に `.gitconfig.local` ファイルが作成されました**
   - 場所：`study-abroad-comparison/.gitconfig.local`

2. **ファイルを編集して、あなたの情報に置き換えてください**
   - `Your Name Here` → あなたのGitHubのユーザー名
   - `your-email@example.com` → あなたのメールアドレス

3. **ファイルの内容を変更**

例：
```
[user]
	name = kiris
	email = b2400560@gl.aiu.ac.jp
```

4. **この設定を有効にする**
   - Cursorを再起動する
   - または、以下のコマンドを実行（Gitがインストールされている場合）

---

## 🔧 方法2: グローバル設定ファイルを手動で作成

すべてのプロジェクトに適用する場合：

### 手順

1. **ホームフォルダに `.gitconfig` ファイルを作成**
   - Windowsの場合：`C:\Users\kiris\.gitconfig`

2. **以下の内容をコピーして保存**

```
[user]
	name = あなたのGitHubのユーザー名
	email = あなたのメールアドレス
```

**例：**
```
[user]
	name = kiris
	email = b2400560@gl.aiu.ac.jp
```

3. **ファイルを保存**

---

## 🔧 方法3: Cursorの設定から直接入力

1. **Cursorの設定を開く**
   - `Ctrl + ,` を押す

2. **検索ボックスで「git config」と入力**

3. **Git設定を検索**
   - 「Git: Path」などGit関連の設定が表示されます

4. **設定画面から直接設定できる場合もあります**

---

## 🔧 方法4: エクスプローラーから手動でファイルを作成

1. **エクスプローラーを開く**

2. **ホームフォルダに移動**
   - `C:\Users\kiris\`

3. **新しいテキストファイルを作成**
   - 右クリック → 「新規作成」→ 「テキストドキュメント」

4. **ファイル名を変更**
   - `.gitconfig` に変更（先頭の点「.」も含む）
   - ⚠️ 拡張子（.txt）を削除する必要があります

5. **ファイルを開いて、以下を入力**

```
[user]
	name = あなたのGitHubのユーザー名
	email = あなたのメールアドレス
```

6. **保存**

---

## ✅ どの方法を使うべきか

- **方法1（プロジェクトローカル）**: このプロジェクトだけに設定したい場合
- **方法2（グローバル）**: すべてのプロジェクトに同じ設定を使いたい場合（推奨）

---

## 📝 設定する情報

- **name**: GitHubのユーザー名、または表示したい名前
  - 例：`kiris`

- **email**: GitHubに登録しているメールアドレス
  - 例：`b2400560@gl.aiu.ac.jp` または `your-email@gmail.com`

---

## ✅ 設定後

設定ファイルを作成・編集したら：

1. **Cursorを再起動**
2. **もう一度コミットを試す**
   - ソース管理パネル（📊）を開く
   - コミットメッセージを入力
   - `Ctrl + Enter` を押す

これでコミットできるはずです！
