# 🚨 Connection Failed エラー → 今すぐやること

## ⚡ 3ステップで解決

### ステップ1: Cursorでターミナルを開く

1. **Cursorを開く**（既に開いている場合はそのまま）

2. **キーボードで以下を同時に押す：**
   - `Ctrl` キー
   - `Shift` キー
   - ` ` キー（数字の1の左にあるキー）

3. **Cursorの下部に黒い画面が表示される**
   → これが「ターミナル」です

---

### ステップ2: コマンドを実行する

ターミナルが開いたら、以下を**順番に**実行してください：

#### 1つ目：以下の文字をコピーして貼り付け、Enterキーを押す

```
cd C:\Users\kiris\.cursor\study-abroad-comparison
```

#### 2つ目：以下の文字をコピーして貼り付け、Enterキーを押す

```
npm run dev
```

---

### ステップ3: 起動を待つ

ターミナルに、以下のようなメッセージが表示されるまで待ってください（30秒〜1分）：

```
✓ Ready in X seconds
○ Local:        http://localhost:3000
```

**このメッセージが表示されれば、開発サーバーは起動しています！**

---

### ステップ4: ブラウザで開く

1. **ブラウザを開く**（Chrome、Edge、Firefoxなど）

2. **アドレスバー（URLを入力するところ）に以下を入力：**
   ```
   http://localhost:3000
   ```

3. **Enterキーを押す**

4. **ページが表示されれば成功です！🎉**

---

## ❗ 重要

開発サーバーを起動した後は、**ターミナルを閉じないでください**。

ターミナルを閉じると、開発サーバーも停止してしまいます。

**サーバーを停止したい場合**は、ターミナルで `Ctrl + C` を押してください。

---

## ❓ うまくいかない場合

### 「npm が見つかりません」というエラーが出る

→ Node.jsがインストールされていない可能性があります。
   まず、Node.jsをインストールする必要があります。

### ターミナルが開かない

→ `Ctrl + Shift + ` をもう一度試してください。
   または、Cursorの上部メニューから「ターミナル」→「新しいターミナル」をクリックしてください。

### その他のエラーが出る

→ ターミナルに表示されたエラーメッセージを教えてください。
   そのエラーメッセージを見れば、解決方法が分かります。

---

**まずは、上記の3ステップを試してみてください！**
