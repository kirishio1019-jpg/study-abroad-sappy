# 🚨 緊急対応：プロジェクトを再インポートする手順

## 再デプロイしても変更が反映されない場合の最終手段

### 📋 手順（ステップバイステップ）

#### ステップ1: Vercelダッシュボードにアクセス

**👇 このリンクをクリック：**
https://vercel.com/dashboard

#### ステップ2: プロジェクトを削除

1. プロジェクト「study-abroad-sappy」を開く
2. **「Settings」タブ**をクリック
3. **「General」**を開く
4. 一番下までスクロール
5. **「Delete Project」**ボタンをクリック
6. プロジェクト名「study-abroad-sappy」を入力
7. **「Delete」**をクリックして削除を確認

#### ステップ3: プロジェクトを再インポート

1. **「Add New...」**ボタンをクリック
2. **「Project」**を選択
3. GitHubリポジトリを選択：
   - 「study-abroad-sappy」を探す
   - または検索バーで「study-abroad-sappy」を検索
4. リポジトリを選択して**「Import」**をクリック

#### ステップ4: プロジェクト設定を確認

以下の設定を確認してください：

- ✅ **Project Name**: `study-abroad-sappy`
- ✅ **Root Directory**: **空白**（自動検出）
- ✅ **Framework Preset**: **Next.js**（自動検出されるはず）
- ✅ **Build Command**: 空白（自動検出）または `npm run build`
- ✅ **Output Directory**: 空白（自動検出）または `.next`
- ✅ **Install Command**: 空白（自動検出）または `npm install`

#### ステップ5: 環境変数を設定

**重要：環境変数を設定してください**

1. **「Environment Variables」**セクションを開く
2. 以下の環境変数を追加：

   **変数1:**
   - Name: `NEXT_PUBLIC_SUPABASE_URL`
   - Value: （あなたのSupabaseのURL）
   - Environment: **すべてにチェック**（Production, Preview, Development）

   **変数2:**
   - Name: `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - Value: （あなたのSupabaseのAnon Key）
   - Environment: **すべてにチェック**（Production, Preview, Development）

3. **「Add」**をクリックして各変数を追加

#### ステップ6: デプロイを開始

1. 設定を確認
2. **「Deploy」**ボタンをクリック
3. 数分待つ（ビルドが完了するまで）

#### ステップ7: デプロイ完了を確認

1. デプロイが完了したら、**デプロイURL**が表示されます
2. そのURLにアクセスして、アプリが正常に動作しているか確認
3. 変更が反映されているか確認

## 🔗 必要なリンク

- **Vercelダッシュボード**: https://vercel.com/dashboard
- **GitHubリポジトリ**: https://github.com/kirishio1019-jpg/study-abroad-sappy
- **Supabaseダッシュボード**: https://supabase.com/dashboard（環境変数を確認するため）

## ✅ 確認事項

- [ ] プロジェクトを削除できた
- [ ] プロジェクトを再インポートできた
- [ ] Root Directoryが空白になっている
- [ ] 環境変数を設定した（2つとも）
- [ ] すべての環境（Production, Preview, Development）に環境変数を設定した
- [ ] デプロイが完了した
- [ ] デプロイURLにアクセスできた
- [ ] アプリが正常に動作している
- [ ] 変更が反映されている

## 💡 ヒント

- 環境変数の値は、Supabaseダッシュボードの「Settings → API」から取得できます
- デプロイには数分かかります。完了するまで待ってください
- デプロイが完了したら、ブラウザでデプロイURLにアクセスしてください
