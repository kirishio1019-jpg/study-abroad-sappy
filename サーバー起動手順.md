# 開発サーバーの起動手順（詳細版）

## 🚨 重要: まず確認すること

### ステップ1: 開発サーバーが起動しているか確認

ブラウザで以下のURLを開いてください：
```
http://localhost:3000
```

**もし「このサイトにアクセスできません」というエラーが表示される場合、開発サーバーが起動していません。**

---

## 📋 開発サーバーを起動する方法

### 方法1: バッチファイルを使用（最も簡単）

1. **ファイルエクスプローラーを開く**
   - Windowsキー + E を押す
   - または、エクスプローラーを手動で開く

2. **以下のパスに移動:**
   ```
   C:\Users\kiris\.cursor\study-abroad-comparison
   ```

3. **`開発サーバー起動.bat` をダブルクリック**
   - コマンドプロンプトウィンドウが開きます
   - 以下のようなメッセージが表示されるまで待ってください：
     ```
     ✓ Ready in X seconds
     ○ Local:        http://localhost:3000
     ```

4. **コマンドプロンプトウィンドウは開いたままにしてください**
   - このウィンドウを閉じると、開発サーバーも停止します

5. **ブラウザで `http://localhost:3000` を開く**

---

### 方法2: Cursorのターミナルを使用

1. **Cursorを開く**

2. **ターミナルを開く**
   - キーボードショートカット: `Ctrl + Shift + ` (バッククォート)
   - または、上部メニューから「ターミナル」→「新しいターミナル」

3. **ターミナルに以下を入力（1行ずつEnterキーを押す）:**
   ```powershell
   cd C:\Users\kiris\.cursor\study-abroad-comparison
   ```
   
   その後:
   ```powershell
   npm run dev
   ```

4. **起動メッセージを確認**
   - 以下のようなメッセージが表示されるまで待ってください：
     ```
     ✓ Ready in X seconds
     ○ Local:        http://localhost:3000
     ```

5. **ブラウザで `http://localhost:3000` を開く**

---

## 🔍 トラブルシューティング

### 問題1: バッチファイルをダブルクリックしても何も起きない

**解決方法:**
1. 右クリック → 「管理者として実行」を試す
2. または、方法2（Cursorのターミナル）を使用する

---

### 問題2: 「npm が見つかりません」というエラー

**解決方法:**
1. Node.jsがインストールされているか確認:
   - コマンドプロンプトまたはPowerShellを開く
   - 以下を実行:
     ```powershell
     node --version
     npm --version
     ```
2. インストールされていない場合:
   - https://nodejs.org/ からインストール
   - インストール後、コンピューターを再起動

---

### 問題3: 「Port 3000 is already in use」というエラー

**解決方法:**
1. 別のポートを使用:
   ```powershell
   npm run dev -- -p 3001
   ```
2. その後、`http://localhost:3001` でアクセス

---

### 問題4: 「Cannot find module」というエラー

**解決方法:**
1. 依存関係をインストール:
   ```powershell
   cd C:\Users\kiris\.cursor\study-abroad-comparison
   npm install
   ```
2. インストールが完了したら、再度開発サーバーを起動:
   ```powershell
   npm run dev
   ```

---

### 問題5: コマンドプロンプトがすぐに閉じてしまう

**解決方法:**
1. バッチファイルを右クリック → 「編集」
2. 最後の行に以下を追加:
   ```bat
   pause
   ```
3. 保存して再度実行

---

## ✅ 正常に起動したか確認する方法

1. **コマンドプロンプト/ターミナルに以下のメッセージが表示されている:**
   ```
   ✓ Ready in X seconds
   ○ Local:        http://localhost:3000
   ```

2. **ブラウザで `http://localhost:3000` を開いたとき:**
   - ページが表示される（エラーではなく、何らかのコンテンツが表示される）

3. **もしテストページにアクセスできる場合:**
   ```
   http://localhost:3000/test.html
   ```
   - これも表示されれば、サーバーは正常に動作しています

---

## 🛑 開発サーバーを停止する方法

- コマンドプロンプト/ターミナルウィンドウで `Ctrl + C` を押す
- または、ウィンドウを閉じる

---

## 📞 まだ解決しない場合

以下の情報を教えてください：

1. **どの方法で起動しようとしましたか？**
   - バッチファイル / Cursorのターミナル / その他

2. **エラーメッセージは何でしたか？**
   - エラーメッセージの全文をコピーしてください

3. **ブラウザで `http://localhost:3000` を開いたとき、何が表示されましたか？**
   - 「このサイトにアクセスできません」
   - 白い画面
   - ローディングが続く
   - その他のメッセージ
