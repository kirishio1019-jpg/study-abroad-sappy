# Vercelのリンク（URL）について

## 📍 基本ルール

**同じリンクで内容が更新されます！**

## 🔗 リンクの種類

### 1. プロダクションURL（メインのリンク）

**このリンクは変わりません：**

```
https://study-abroad-sappy.vercel.app
```

または

```
https://study-abroad-sappy-[あなたのユーザー名].vercel.app
```

**特徴：**
- ✅ このURLは**ずっと同じ**です
- ✅ 新しい変更をデプロイすると、**同じURLで内容が更新**されます
- ✅ これが本番環境（Production）のURLです

### 2. プレビューデプロイURL

**各デプロイごとに新しいURLが生成されます：**

```
https://study-abroad-sappy-xxxxx.vercel.app
```

**特徴：**
- 🔄 各デプロイごとに**新しいURLが生成**されます
- 🔍 これはプレビュー環境用のURLです
- 📝 コードをpushするたびに、新しいプレビューURLが作成されます

## 💡 わかりやすく説明すると

### 例：

1. **最初のデプロイ**
   - URL: `https://study-abroad-sappy.vercel.app`
   - 内容: バージョン1

2. **変更をデプロイ後**
   - URL: `https://study-abroad-sappy.vercel.app` ← **同じURL！**
   - 内容: バージョン2 ← **内容が更新される**

3. **さらに変更をデプロイ後**
   - URL: `https://study-abroad-sappy.vercel.app` ← **同じURL！**
   - 内容: バージョン3 ← **内容が更新される**

## 📋 確認方法

### Vercelダッシュボードで確認：

1. **Vercelダッシュボードにアクセス**
   - https://vercel.com/dashboard
   - プロジェクト「study-abroad-sappy」を開く

2. **プロダクションURLを確認**
   - プロジェクトページの上部に表示されているURL
   - これが**常に同じメインのURL**です

3. **プレビューデプロイURLを確認**
   - 「Deployments」タブを開く
   - 各デプロイに独自のURLが表示されます
   - これらは**各デプロイごとに変わる**プレビューURLです

## ✅ まとめ

- **メインのURLは変わりません** → 同じリンクを使い続けられます
- **内容は自動的に更新されます** → 新しいデプロイをすると、同じURLで最新の内容が表示されます
- **プレビューURLは変わります** → 各デプロイごとに新しいURLが生成されます（これは本番環境とは別）

## 🔍 実際の動作

1. コードを変更してpushする
2. Vercelが自動的にデプロイを開始
3. デプロイが完了すると、**同じURLで新しい内容が表示される**
4. ブラウザでアクセスすると、最新の内容が表示される

**注意：** ブラウザのキャッシュの影響で、古い内容が表示されることがあります。その場合は、ページを再読み込み（Ctrl+F5 または Cmd+Shift+R）してください。
