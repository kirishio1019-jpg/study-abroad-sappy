# 🚀 開発サーバーを起動する方法（初めての方向け）

## このページを読んで、順番に実行してください

---

## ステップ1: Cursorでターミナルを開く

### 方法A: キーボードショートカット（簡単）

1. **Cursorを開く**

2. **以下のキーを同時に押す:**
   - `Ctrl` キー
   - `Shift` キー  
   - ` ` キー（数字の1の左にあるキー）

   **同時に押すと、Cursorの下部に黒い画面が表示されます**
   
   これが「ターミナル」です。

### 方法B: メニューから開く

1. **Cursorの上部にあるメニューバーを見る**

2. **「ターミナル」という文字をクリック**

3. **「新しいターミナル」をクリック**

4. **Cursorの下部に黒い画面が表示されます**

---

## ステップ2: ターミナルが開いたことを確認する

Cursorの**下部**に、以下のような黒い（または白い）画面が表示されていればOKです：

```
PS C:\Users\kiris\.cursor> 
```

このような画面が表示されていれば、ターミナルは正常に開いています。

---

## ステップ3: コマンドを実行する

### 1つ目のコマンド:

ターミナルの画面をクリックして、以下の文字を**そのままコピーして貼り付け**してください：

```
cd C:\Users\kiris\.cursor\study-abroad-comparison
```

貼り付けたら、**Enterキーを押してください**

### 2つ目のコマンド:

次に、以下の文字を**そのままコピーして貼り付け**してください：

```
npm run dev
```

貼り付けたら、**Enterキーを押してください**

---

## ステップ4: 起動を待つ

ターミナルに、以下のようなメッセージが表示されるまで待ってください（30秒〜1分くらいかかります）：

```
✓ Ready in X seconds
○ Local:        http://localhost:3000
```

**このメッセージが表示されれば、開発サーバーは起動しています！**

---

## ステップ5: ブラウザで開く

1. **ブラウザを開く**（Chrome、Edge、Firefoxなど、どれでもOK）

2. **アドレスバー（URLを入力するところ）をクリック**

3. **以下の文字を入力:**
   ```
   http://localhost:3000
   ```

4. **Enterキーを押す**

5. **ページが表示されれば成功です！**

---

## ❗ 重要: ターミナルは開いたままにしておく

開発サーバーを起動した後は、**ターミナルを閉じないでください**。

ターミナルを閉じると、開発サーバーも停止してしまいます。

---

## 🛑 開発サーバーを停止する方法

開発サーバーを停止したい場合は：

1. **ターミナルをクリック**
2. **`Ctrl + C` を押す**（CtrlキーとCキーを同時に押す）

---

## 📞 困ったときは

### Q: ターミナルが表示されません

A: 
- `Ctrl + Shift + ` をもう一度押してみてください
- または、Cursorの上部メニューから「ターミナル」→「新しいターミナル」をクリック

### Q: コマンドを入力する場所が分かりません

A:
- Cursorの下部に黒い（または白い）画面があるはずです
- その画面の中に、点滅しているカーソル（|）があります
- そこをクリックしてから、コマンドを貼り付けてください

### Q: 「npm が見つかりません」というエラーが出ます

A:
- Node.jsがインストールされていない可能性があります
- この場合は、まずNode.jsをインストールする必要があります
