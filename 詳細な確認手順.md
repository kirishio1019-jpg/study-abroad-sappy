# 開発サーバーが開けない問題 - 詳細な確認手順

## 現在の状況を確認してください

### 質問1: どのような方法で開発サーバーを起動しようとしましたか？

- [ ] バッチファイル（`開発サーバー起動.bat`）をダブルクリック
- [ ] Cursorのターミナルで `npm run dev` を実行
- [ ] その他の方法

---

### 質問2: 開発サーバーを起動したとき、何が表示されましたか？

#### パターンA: コマンドプロンプトまたはターミナルにメッセージが表示された
どのようなメッセージでしたか？以下から選んでください：
- [ ] `✓ Ready in X seconds` と表示された
- [ ] `○ Local: http://localhost:3000` と表示された
- [ ] エラーメッセージが表示された（エラー内容を教えてください）
- [ ] 何も表示されなかった
- [ ] コマンドプロンプト/ターミナルが開かなかった

#### パターンB: コマンドプロンプト/ターミナルが開かなかった
- バッチファイルをダブルクリックしても何も起きなかった
- → Windowsのセキュリティ設定の問題かもしれません

---

### 質問3: ブラウザで `http://localhost:3000` にアクセスしたとき、何が表示されましたか？

- [ ] **ページが正常に表示された**
- [ ] **「このサイトにアクセスできません」** または「接続できません」というエラー
- [ ] **白い画面**（何も表示されない）
- [ ] **ローディングがずっと続く**（回転するアイコンなど）
- [ ] **エラーページ**（404エラーなど）
- [ ] **その他のメッセージ**（内容を教えてください）

---

### 質問4: ブラウザの開発者ツール（F12）を開いたとき、何が表示されましたか？

1. ブラウザで `F12` キーを押す
2. **Console** タブを開く
3. エラーメッセージがあれば、その内容を教えてください

---

## すぐに試せる解決方法

### 解決方法1: テストHTMLファイルにアクセス

1. 開発サーバーが起動していることを確認
2. ブラウザで以下のURLを開く：
   ```
   http://localhost:3000/test.html
   ```
3. もしこのページが表示されれば、開発サーバーは正常に動作しています

---

### 解決方法2: ポートを変更して起動

ポート3000が使用中の可能性があります。別のポートを使用してみてください：

1. ターミナルで以下を実行：
   ```powershell
   cd C:\Users\kiris\.cursor\study-abroad-comparison
   npm run dev -- -p 3001
   ```
2. ブラウザで `http://localhost:3001` を開く

---

### 解決方法3: 依存関係を再インストール

1. ターミナルで以下を実行：
   ```powershell
   cd C:\Users\kiris\.cursor\study-abroad-comparison
   npm install
   ```
2. インストールが完了したら、開発サーバーを起動：
   ```powershell
   npm run dev
   ```
