# デプロイURLの確認方法

## 🔍 デプロイURLを確認する方法

### 方法1: Vercelダッシュボードで確認（最も確実）

1. **Vercelダッシュボードにアクセス**
   - https://vercel.com/dashboard

2. **プロジェクトを開く**
   - プロジェクト一覧から「**study-abroad-sappy**」をクリック

3. **デプロイURLを確認**
   - プロジェクトページの**上部**に表示されているURLがデプロイURLです
   - 通常、以下のような形式です：
     - `https://study-abroad-sappy.vercel.app`
     - または `https://study-abroad-sappy-[あなたのユーザー名].vercel.app`

### 方法2: 最新のデプロイから確認

1. **Vercelダッシュボードにアクセス**
   - https://vercel.com/dashboard

2. **プロジェクト「study-abroad-sappy」を開く**

3. **「Deployments」タブを開く**

4. **最新のデプロイをクリック**

5. **デプロイURLを確認**
   - デプロイの詳細ページに表示されているURL

## 🔗 一般的なデプロイURLの形式

プロジェクト名は「study-abroad-sappy」なので、デプロイURLは通常以下の形式です：

```
https://study-abroad-sappy.vercel.app
```

または

```
https://study-abroad-sappy-[あなたのユーザー名].vercel.app
```

## 📱 スマホでキャッシュをクリアしてアクセスする方法

デプロイURLが確認できたら：

1. **スマホのブラウザでデプロイURLにアクセス**
2. **キャッシュをクリアしたい場合**：
   - デプロイURLに `?clear_cache=true` を追加
   - 例: `https://study-abroad-sappy.vercel.app?clear_cache=true`
3. **ページをリロード**
   - 画面を下に引っ張って更新（プルダウンリフレッシュ）
