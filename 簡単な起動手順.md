# 開発サーバーの簡単な起動手順

## 🎯 最も簡単な方法：Cursorのターミナルを使う

### ステップ1: Cursorでターミナルを開く

1. **Cursorを開く**（既に開いている場合はそのまま）

2. **キーボードのショートカットを押す:**
   - `Ctrl + Shift + ` 
   - 最後の` `は、数字の1の左にあるキー（バッククォート）です
   
   または

3. **Cursorの上部メニューから開く:**
   - メニューバーの「ターミナル」をクリック
   - 「新しいターミナル」をクリック

### ステップ2: ターミナルが表示されたら

Cursorの**下部**に、黒い画面（または白い画面）が表示されます。
これが「ターミナル」です。

### ステップ3: コマンドを入力する

ターミナルに、以下の文字を**そのままコピーして貼り付け**してください：

```powershell
cd C:\Users\kiris\.cursor\study-abroad-comparison
```

**Enterキーを押します**

次に、以下の文字を**そのままコピーして貼り付け**してください：

```powershell
npm run dev
```

**Enterキーを押します**

### ステップ4: 起動を待つ

ターミナルに、以下のようなメッセージが表示されるまで待ってください（30秒〜1分くらいかかります）：

```
✓ Ready in X seconds
○ Local:        http://localhost:3000
```

### ステップ5: ブラウザで開く

1. **ブラウザを開く**（Chrome、Edge、Firefoxなど、どれでもOK）

2. **アドレスバー（URLを入力するところ）に以下を入力:**
   ```
   http://localhost:3000
   ```

3. **Enterキーを押す**

4. **ページが表示されれば成功です！**

## 📸 ターミナルの見た目

ターミナルは、Cursorの**下部**に表示される黒い（または白い）画面で、以下のような感じです：

```
PS C:\Users\kiris\.cursor> 
```

このような画面が表示されれば、それがターミナルです。

## ❓ よくある質問

### Q: ターミナルが表示されません

A: 
- `Ctrl + Shift + ` をもう一度押してみてください
- 最後の` `は、数字の1の左にあるキーです
- または、Cursorの上部メニューから「ターミナル」→「新しいターミナル」をクリック

### Q: コマンドを入力するところが見つかりません

A:
- Cursorの下部に黒い（または白い）画面があるはずです
- その画面の中に、点滅しているカーソル（|）があるはずです
- そこに文字を入力できます

### Q: 「npm が見つかりません」というエラーが出ます

A:
- Node.jsがインストールされていない可能性があります
- この場合は、まずNode.jsをインストールする必要があります

### Q: 「Port 3000 is already in use」というエラーが出ます

A:
- 既に開発サーバーが起動している可能性があります
- ブラウザで `http://localhost:3000` を開いてみてください
- もしくは、別のポートを使用：
  ```powershell
  npm run dev -- -p 3001
  ```
  その後、`http://localhost:3001` でアクセス
