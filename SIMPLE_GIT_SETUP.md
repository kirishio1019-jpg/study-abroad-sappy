# Git設定を簡単に設定する方法

## 🎯 最も簡単な方法：エクスプローラーで設定ファイルを作成

### ステップ1: エクスプローラーを開く

1. **Windowsキー + E** を押してエクスプローラーを開く

2. **アドレスバーに以下のパスを入力してEnter**
   ```
   C:\Users\kiris
   ```
   （または、ホームフォルダに直接移動）

---

### ステップ2: 設定ファイルを作成

1. **空白部分で右クリック**
   - 「新規作成」→ 「テキストドキュメント」を選択

2. **ファイル名を変更**
   - ファイル名を `.gitconfig` に変更
   - ⚠️ **重要**: 先頭に点「.」をつけて、拡張子（.txt）を削除
   - ファイル名は `.gitconfig` のみ（拡張子なし）

3. **「ファイル名を変更すると、ファイルが使えなくなる可能性があります」という警告が出たら**
   - 「はい」をクリック

---

### ステップ3: ファイルを編集

1. **`.gitconfig` ファイルを右クリック**
   - 「プログラムから開く」→ 「メモ帳」（または「Notepad」）

2. **以下の内容をコピー＆ペースト**
   ```
   [user]
   	name = あなたのGitHubのユーザー名
   	email = あなたのメールアドレス
   ```

3. **あなたの情報に置き換える**
   - `あなたのGitHubのユーザー名` → 実際のユーザー名（例：`kiris`）
   - `あなたのメールアドレス` → 実際のメールアドレス（例：`b2400560@gl.aiu.ac.jp`）

4. **保存**
   - `Ctrl + S` を押す
   - ファイルを閉じる

---

### ステップ4: Cursorで確認

1. **Cursorを再起動**（完全に閉じて再度開く）

2. **ソース管理パネル（📊）を開く**

3. **コミットメッセージを入力して `Ctrl + Enter`**

これでコミットできるはずです！

---

## 📝 例

`.gitconfig`ファイルの内容例：

```
[user]
	name = kiris
	email = b2400560@gl.aiu.ac.jp
```

## 🆘 ファイル名が変更できない場合

**方法A: メモ帳で保存時に変更**

1. メモ帳を開く
2. 内容を入力：
   ```
   [user]
   	name = あなたの名前
   	email = あなたのメール
   ```
3. 「名前を付けて保存」（`Ctrl + Shift + S`）
4. 「ファイルの種類」を「すべてのファイル (*.*)」に変更
5. ファイル名を `.gitconfig` と入力（引用符なし）
6. 保存場所を `C:\Users\kiris\` に指定
7. 「保存」をクリック

**方法B: コマンドプロンプトから作成**

1. `Win + R` を押す
2. `cmd` と入力してEnter
3. 以下のコマンドを実行（あなたの情報に置き換えて）：

```cmd
cd C:\Users\kiris
echo [user] > .gitconfig
echo 	name = あなたの名前 >> .gitconfig
echo 	email = あなたのメール >> .gitconfig
```

## ✅ 確認方法

設定が完了したら、Cursorでコミットを試してみてください！

もしエラーが出たら、以下を確認：
- ✅ ファイル名が正確に `.gitconfig` になっているか（拡張子なし）
- ✅ ファイルが `C:\Users\kiris\` に保存されているか
- ✅ 内容が正しい形式になっているか
- ✅ Cursorを再起動したか
