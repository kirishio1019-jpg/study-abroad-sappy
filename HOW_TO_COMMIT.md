# コミットメッセージの入力欄が表示されない場合

## 🔍 確認事項

### 1. ファイルがステージングされているか確認

ソース管理パネル（📊）を開いたときに、以下が表示されます：

- **「Changes」セクション** - まだステージングされていないファイル
- **「Staged Changes」セクション** - ステージング済みのファイル

**コミットメッセージ入力欄が表示される条件：**
- ✅ **「Staged Changes」にファイルがある**こと

---

## 📝 手順

### ステップ1: ファイルをステージングする

1. **ソース管理パネル（📊）を開く**
   - 左側のメニューで 📊 アイコンをクリック
   - または `Ctrl + Shift + G`

2. **「Changes」セクションを確認**
   - 変更されたファイルが表示されます

3. **すべてのファイルをステージング**
   - 方法A: 「**+**」アイコンの横にある「**Stage All Changes**」をクリック
   - 方法B: コマンドパレット（`Ctrl + Shift + P`）→ 「**Git: Stage All Changes**」と入力
   - 方法C: 各ファイルの横にある「**+**」ボタンをクリック

### ステップ2: 「Staged Changes」にファイルが移動したことを確認

- ✅ 「Changes」セクションが空になる
- ✅ 「Staged Changes」セクションにファイルが表示される

### ステップ3: コミットメッセージ入力欄が表示される

「Staged Changes」にファイルがあると、**上部にテキストボックスが表示されます**：

```
┌─────────────────────────────────────┐
│  📊 SOURCE CONTROL                  │
├─────────────────────────────────────┤
│                                     │
│  📝 Message (or press Ctrl+Enter...)│ ← ここ！
│  ┌───────────────────────────────┐ │
│  │                               │ │
│  └───────────────────────────────┘ │
│                                     │
│  Staged Changes                     │
│  ┌───────────────────────────────┐ │
│  │ ✓ 📄 file1.ts                 │ │
│  │ ✓ 📄 file2.tsx                │ │
│  └───────────────────────────────┘ │
│                                     │
└─────────────────────────────────────┘
```

### ステップ4: コミット

1. **テキストボックスにコミットメッセージを入力**
   - 例：`Initial commit: Sappy Study Abroad Platform`

2. **コミットを実行**
   - `Ctrl + Enter` を押す
   - または、テキストボックスの上にある「**✓ Commit**」ボタンをクリック

---

## 🆘 それでも表示されない場合

### 方法1: コマンドパレットからコミット

1. `Ctrl + Shift + P` を押す
2. 「**Git: Commit**」と入力して選択
3. コミットメッセージを入力するポップアップが表示されます
4. メッセージを入力してEnter

### 方法2: ファイルがステージングされているか再確認

1. ソース管理パネルで「Staged Changes」セクションを確認
2. ファイルがなければ、再度ステージングしてください

---

## ✅ 確認チェックリスト

- [ ] ソース管理パネル（📊）を開いた
- [ ] 「Changes」にファイルが表示されている
- [ ] 「Stage All Changes」をクリックした
- [ ] 「Staged Changes」にファイルが移動した
- [ ] 上部にテキストボックスが表示されている

すべてチェックできたら、コミットメッセージを入力して `Ctrl + Enter` です！
