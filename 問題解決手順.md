# 開発サーバーが開けない問題の解決手順

## 現在の状況確認

### ステップ1: 開発サーバーが起動しているか確認

1. **バッチファイルを実行**
   - `C:\Users\kiris\.cursor\study-abroad-comparison` フォルダを開く
   - `開発サーバー起動.bat` をダブルクリック
   - コマンドプロンプトウィンドウが開きます

2. **以下のメッセージが表示されるまで待つ:**
   ```
   ✓ Ready in X seconds
   ○ Local:        http://localhost:3000
   ```

3. **もしエラーが出た場合:**
   - エラーメッセージの内容をメモしてください
   - よくあるエラー:
     - `Port 3000 is already in use` → ポート3000が使用中
     - `Cannot find module` → `npm install` を実行する必要があります

---

### ステップ2: ブラウザでアクセス

1. **ブラウザを開く**
   - Chrome、Edge、Firefoxなど、どれでもOK

2. **アドレスバーに以下を入力:**
   ```
   http://localhost:3000
   ```
   または
   ```
   localhost:3000
   ```

3. **Enterキーを押す**

---

### ステップ3: 何が表示されるか確認

#### ✅ 正常な場合
- ページが表示される
- 「Sappy - Study Abroad Platform」のナビゲーションバーが見える

#### ❌ 問題がある場合

**パターンA: 「このサイトにアクセスできません」**
- → 開発サーバーが起動していない可能性
- バッチファイルを実行して、エラーが出ていないか確認

**パターンB: 白い画面（何も表示されない）**
- → ブラウザの開発者ツール（F12）を開く
- Consoleタブでエラーメッセージを確認

**パターンC: ローディングが続く**
- → ブラウザの開発者ツール（F12）を開く
- Consoleタブでエラーメッセージを確認
- Networkタブでリクエストが失敗していないか確認

---

## トラブルシューティング

### 問題1: バッチファイルを実行しても何も起きない

**解決方法:**
1. 右クリック → 「管理者として実行」を試す
2. または、PowerShellを開いて以下を実行:
   ```powershell
   cd C:\Users\kiris\.cursor\study-abroad-comparison
   npm run dev
   ```

---

### 問題2: 「npm が見つかりません」というエラー

**解決方法:**
1. Node.jsがインストールされているか確認
2. コマンドプロンプトで以下を実行:
   ```powershell
   node --version
   npm --version
   ```
3. インストールされていない場合、以下からインストール:
   https://nodejs.org/

---

### 問題3: ポート3000が使用中

**解決方法:**
1. 別のポートを使用:
   ```powershell
   npm run dev -- -p 3001
   ```
2. その後、`http://localhost:3001` でアクセス

---

### 問題4: ブラウザでエラーが表示される

**解決方法:**
1. ブラウザで F12 キーを押して開発者ツールを開く
2. Consoleタブを開く
3. 表示されているエラーメッセージを確認
4. エラーメッセージをメモして、次の情報と一緒に報告:
   - どのステップで問題が発生したか
   - 表示されているエラーメッセージ
   - ブラウザの種類（Chrome、Edgeなど）

---
